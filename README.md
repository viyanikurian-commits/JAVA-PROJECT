# JAVA-PROJECT
JAVA PROJECT- 4th SEM 
TRAVEL EXPENSE TRACKER 
This application helps travelers keep track of their spending, calculate total trip expenses, and stay within their budget. It allows users to record daily expenses such as accommodation, transportation, food, shopping, and entertainment, providing a clear overview of where their money is going. The app automatically calculates the total trip cost, compares it with the planned budget, and helps users manage their finances more effectively while traveling.
